Thank you for installing {{ .Chart.Name }}.

To verify the deployment, you can run the Helm tests:
  $ helm test {{ .Release.Name }}

To access the services:

{{- if .Values.mlflow.enabled }}
1. MLflow:
   {{- if eq .Values.mlflow.service.type "LoadBalancer" }}
   $ kubectl get svc {{ .Release.Name }}-mlflow --namespace {{ .Release.Namespace }} -o jsonpath="{.status.loadBalancer.ingress[0].ip}"
   {{- else if eq .Values.mlflow.service.type "ClusterIP" }}
   $ kubectl port-forward svc/{{ .Release.Name }}-mlflow {{ .Values.mlflow.service.port }}:{{ .Values.mlflow.service.port }}
   {{- end }}
   MLflow UI will be available at: http://localhost:{{ .Values.mlflow.service.port }}
{{- end }}
